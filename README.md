# SMART Health Card Demo

A demonstration of implementing SMART Health Cards using Medplum and Kill the Clipboard library, showcasing the FHIR Operation `$health-cards-issue` for generating verifiable health credentials.

## Resources

- **FHIR Operation $health-cards-issue**: [HL7 Specification](https://hl7.org/fhir/uv/smart-health-cards-and-links/OperationDefinition-patient-i-health-cards-issue.html)
- **Kill the Clipboard Library**: [GitHub Repository](https://github.com/vintasoftware/kill-the-clipboard)
- **Medplum**: [GitHub Repository](https://github.com/medplum/medplum)
